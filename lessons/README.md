# Lessons

`lessons/` は exercise の答え合わせ用スクリプトです。

各 lesson は同じ番号の module doc と exercise に対応します。

```text
docs/modules/07_vector_add_first_kernel.md
  -> exercises/07_vector_add_first_kernel.md
  -> lessons/07_vector_add_first_kernel.py
```

最初に lesson だけを読むと、なぜその実装になっているかが見えにくくなります。原則として docs -> exercises -> lessons の順に進めてください。
